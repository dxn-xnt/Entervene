# Teacher View Features - Quick Reference

## 🎯 Where to Find New Features

### 1. View & Download Files
**Location**: Any Classwork → Attachments Section
- Files show with type icons (📄 PDF, 📋 Word, 🖼️ Image, etc.)
- Shows file size (KB or MB)
- 👁️ Open button - view file immediately
- ⬇️ Download button - save file to device

### 2. Edit Classwork
**Location**: Any Classwork You Created → ✏️ Edit Button (top-right)
**Can Edit**:
- Title, Description, Instructions
- Type & Category
- Points value
- Files (add or remove)

### 3. Monitor Student Submissions
**Location**: Any Classwork → Student Submissions Section
**Shows**:
- How many submitted (✅)
- How many haven't submitted (❌)
- Total students (👥)

**Details Per Student**:
- Name & submission time
- Files submitted
- Grade (if given)
- Attempt number
- Status badge

### 4. Grade Submissions
**Location**: Classwork → Student Submissions → Click Student Name
- View their files
- See submission time & details
- Add grade and feedback

---

## 📋 Step-by-Step Guides

### Add More Files to Existing Classwork
1. Open the classwork
2. Tap ✏️ Edit button
3. Scroll to "Add More Files"
4. Tap "Pick Files (4MB max)"
5. Select new PDF/documents
6. Tap "Save Changes"
✅ New files are now visible to students

### Download Student Submission
1. Open classwork
2. Scroll to "Student Submissions"
3. Tap the student name
4. Find their files in the submission
5. Tap ⬇️ Download icon
✅ File saved to device

### Check Who's Missing Work
1. Open classwork
2. Scroll to "Student Submissions"
3. Look at "Not Submitted" section (red)
4. See list of students who haven't submitted yet
✅ Quick overview of missing work

### Grade a Submission
1. Open classwork
2. Scroll to "Student Submissions"
3. Tap student in "Submitted" section (green)
4. View their work
5. Add grade and feedback
6. Save
✅ Grade is recorded and student notified

## 🆘 Troubleshooting

**Can't see Edit button?**
- You must be the teacher who created the classwork
- Check if you have proper permissions

**File won't open?**
- Device may not have app for that file type
- Try downloading instead
- Check file isn't corrupted

**Submissions not showing?**
- Try scrolling down to submissions section
- Refresh page if just assigned
- Check students are in the class

**Can't add files?**
- Max 4MB per file
- PDF, Word, PowerPoint, images supported
- Try smaller file first
